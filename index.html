Задания по работе с Promise

Задание 1: Promise с resolve или reject

Этот код создает Promise, который в зависимости от переменной делает resolve или reject:


const condition = true; // измените на false, чтобы увидеть reject

const myPromise = new Promise((resolve, reject) => {
    if (condition) {
        resolve('Promise resolved successfully!');
    } else {
        reject('Promise was rejected.');
    });

Задание 2: Promise с отловом rejected и выводом ошибки

Этот код демонстрирует обработку ошибки с помощью catch:


myPromise
    .then(result => {
        console.log(result);
    })
    .catch(error => {
        console.error('Caught error:', error);
    });

Задание 3: Цепочечное выполнение .then и обработка ошибок


const chainPromise = new Promise((resolve, reject) => {
    resolve('First step success');
});

chainPromise
    .then(result => {
        console.log(result);
        return 'Second step success';
    })
    .then(result => {
        console.log(result);
        throw new Error('An error occurred in the chain');
    })
    .catch(error => {
        console.error('Caught error in chain:', error);
    });

Задание 4: Цепочечное выполнение с двумя catch и finally


const complexPromise = new Promise((resolve, reject) => {
    reject('Initial rejection');
});

complexPromise
    .then(result => {
        console.log(result);
    })
    .catch(error => {
        console.error('First catch:', error);
        throw new Error('Error after first catch');
    })
    .catch(error => {
        console.error('Second catch:', error);
    })
    .finally(() => {
        console.log('Finally block executed');
    });
